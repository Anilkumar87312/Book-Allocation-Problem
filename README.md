# Book Allocation Problem

This project contains a C++ solution for the Book Allocation Problem.

## Problem Statement

Given N books and M students:

* Each student must get at least one book.
* Books are allocated in contiguous order.
* Minimize the maximum number of pages assigned to a student.

## Approach

* Binary Search on Answer
* Check feasibility using greedy allocation

## Time Complexity

O(N * log(sum))

## Space Complexity

O(1)

## Language

C++
